# Phonetic Affect Intensity Multilingual

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.4301068.svg)](https://doi.org/10.5281/zenodo.4301068)

The main objective of the software is to analyze the contribution to the prediction of the detection of polarity in
microblogging sources of the phonetic elements (phonestheme) and emotional elements other than valence (arousal / dominance).

## Embedding Environment

Use Python 3.8 for this repository.

```powershell
pip install -r requirements.txt
python -m spacy download es_core_news_md
python -m spacy download en_core_web_md
python -m spacy download fr_core_news_md
```

## Wikipedia Fair-Comparison Profile (ES/EN/FR)

Use these defaults to keep runs comparable:
- `word_max_samples=20000`
- `part_max_samples=5000` (syllable/phoneme)
- all embedding sizes `150D`
- `--verbosity summary`
- plotting/similarity disabled by default for faster runs

Artifacts generated by ES/EN Wikipedia runs use `*_wiki_*` names so legacy files are not overwritten.
Wikipedia loading now uses streaming + capped local subset cache, so it fetches only the requested sample size.
Cached subsets are stored under `data\embedding\cache\` and reused across runs.

## French Embeddings (from scratch)
